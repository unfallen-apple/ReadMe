나의 미래 모습

전문 개발자로서의 나

5년 후, 나는 웹/앱 개발 분야에서 전문성을 갖춘 개발자가 되어 있을 것입니다. 최신 기술 트렌드를 빠르게 습득하고 적용할 수 있는 능력을 갖추고, 복잡한 문제를 효율적으로 해결할 수 있는 개발자가 되어 있을 것입니다.

특히 사용자 경험(UX)을 중요시하는 개발자로서, 기술적으로 뛰어날 뿐만 아니라 사용자의 니즈를 정확히 파악하고 이를 구현할 수 있는 역량을 갖추고 있을 것입니다.

가치 창출자로서의 나

10년 후, 나는 단순히 코드를 작성하는 개발자를 넘어 실질적인 가치를 창출하는 서비스와 제품을 만드는 사람이 되어 있을 것입니다. 사용자들의 일상을 더 편리하게 만들거나 사회적 문제를 해결하는 서비스를 개발하여 긍정적인 영향력을 행사하고 있을 것입니다.

또한 후배 개발자들을 위한 멘토링과 강의 활동을 통해 지식을 공유하고, 개발자 커뮤니티에 기여하는 역할을 하고 있을 것입니다. 경희대 쿠잇콘과 같은 행사에 강연자로 참여하여 나의 경험과 지식을 나누는 모습을 그려봅니다.

지속적 학습자로서의 나

기술이 빠르게 변화하는 IT 분야에서, 나는 항상 새로운 것을 배우고 적용하는 평생 학습자의 자세를 유지할 것입니다. 단순히 트렌드를 따라가는 것이 아니라, 기술의 본질을 이해하고 이를 창의적으로 활용할 수 있는 개발자가 될 것입니다.

개발자로서의 성장뿐만 아니라 인문학, 디자인, 비즈니스 등 다양한 분야의 지식을 습득하여 융합적 사고가 가능한 개발자로 성장할 것입니다. 후배들에게 "끊임없이 배우고 도전하라"는 메시지를 전달하는 롤모델이 되고 싶습니다.

세모톤 프로젝트 STAR-L 기법으로 분석

Situation (상황)

2025년 3월 23일부터 4월 6일까지 약 2주간 진행된 세모톤(세종대학교 모바일 해커톤)에 참가했습니다. 팀원 6명(프론트엔드 3명, 백엔드 3명)과 함께 온라인과 학생회관 3층에서 모여 프로젝트를 진행했습니다. 주제는 "대학생 커뮤니티 앱"으로, 대학생들의 소통과 정보 공유를 위한 모바일 애플리케이션을 개발하는 것이었습니다.

Task (과제)

주어진 기간 내에 대학생 커뮤니티 모바일 애플리케이션을 개발하는 것이 목표였습니다. 저는 팀 내에서 프론트엔드 개발을 담당했으며, 구체적으로 다음과 같은 업무를 맡았습니다:

  • 사용자 인터페이스(UI) 디자인 및 구현
  • 커뮤니티 게시판 화면 개발
  • 사용자 프로필 페이지 구현
  • 백엔드 API와의 연동

Action (행동)

프론트엔드 개발을 위해 Flutter와 Dart 언어를 학습했습니다. 이전에 경험이 없었기 때문에 유튜브 튜토리얼, 공식 문서, 그리고 Flutter 커뮤니티 자료를 참고하며 빠르게 학습했습니다. 구체적인 행동은 다음과 같습니다:

  • Flutter 기초 문법과 위젯 시스템을 3일 동안 집중적으로 학습했습니다.
  • Figma를 사용하여 UI 디자인 초안을 작성하고 팀원들과 공유했습니다.
  • Material Design 가이드라인을 따라 일관된 디자인 시스템을 구축했습니다.
  • 상태 관리를 위해 Provider 패턴을 적용했습니다.
  • 백엔드 개발자와 정기적인 회의를 통해 API 명세를 확인하고 연동 작업을 진행했습니다.
  • Git을 통한 버전 관리와 GitHub Actions를 활용한 CI/CD 파이프라인을 구축했습니다.
  • 팀원들과 코드 리뷰를 통해 코드 품질을 유지했습니다.

Result (결과)

2주간의 개발 기간 동안 다음과 같은 결과를 얻었습니다:

  • 기본 기능이 구현된 대학생 커뮤니티 앱을 성공적으로 개발했습니다.
  • 사용자 인터페이스는 직관적이고 반응형으로 구현되었으며, 다양한 화면 크기에 대응할 수 있도록 설계되었습니다.
  • 백엔드 API와 성공적으로 연동되어 실시간 데이터를 주고받을 수 있게 되었습니다.
  • 해커톤 발표에서 UI/UX 디자인과 사용자 경험 측면에서 높은 평가를 받았습니다.
  • 팀 프로젝트 우수상을 수상했습니다.
  • GitHub 저장소에 코드를 공개하여 다른 개발자들이 참고할 수 있도록 했습니다.

Learning (배움)

이 프로젝트를 통해 다음과 같은 중요한 교훈을 얻었습니다:

  • 새로운 프로그래밍 언어와 프레임워크를 빠르게 학습하는 방법을 배웠습니다. 필요에 따라 효율적으로 지식을 습득하고 적용하는 능력이 향상되었습니다.
  • 팀 프로젝트에서 효과적인 의사소통의 중요성을 체득했습니다. 정기적인 회의와 명확한 역할 분담이 프로젝트 성공의 핵심이었습니다.
  • Git을 통한 협업 방식과 브랜치 전략의 중요성을 이해하게 되었습니다.
